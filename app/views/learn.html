Create a Device

When you access the dashboard for the first time you don't have any device. To create one, click the create device link and fill in the form.

  • Name - Device name
  • Type - Device type defining the device structure
  • Physical - Physical URI related to your device (leave it empty to simulate the device)
Create device Type selection

When you click on type selection, you are redirected to the type selection page. Here you can pick one of the most popular types or create your types. Once you have decided the type for your device, just click on it.

Types lists Details link

If you want to check the type definitions click on the details link. A detailed definition will show up, allowing you to see the type structure and a simple device simulation. To select the type click on the select this type link.

Selection type link

Do I need the physical device URI?

If you want to connect the physical world, you'll need to fill the physical field. Here you can find an introductive article showing you how to create your first connected device. In case you do not have a physical device to connect do not worry, you can leave the field empty and simulate the device behaviour.
